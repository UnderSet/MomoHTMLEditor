## Build

Open this in Visual Studio. Hit Build. Done.

Alternatively, if you don't fancy VS much or is on macOS/Linux - .NET 10 *SDK* required - `dotnet build`. Done.

*Only tested on Windows.* I'll test on Linux once this reaches a more mature state. macOS might come at some point, but at the moment I don't have a Mac.
